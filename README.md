# YouGe.Core
技术框架 
	前后端分离 
		前端使用VUE
		后端使用 net core 3.1 平台 

数据库 
	使用mysql 
	
技术栈
	AOP 切片编程  日志，请求拦截切片
	IOC 依赖注入 使用 autofac 
	Nlog 日志
	JWT token认证
	identityService4 授权
	Redis 缓存
	Rabbit MQ 消息机制
	swagger
	entity framework ORM
	OSS 存储
	influxDB 时序监控

部署
    kestrel + nginx 反向代理
	linux + Docker
